JAVA COMPILE AND RUN

Execution of java program is done in two stages :

In the above figure they are three files file1.java, file2.java, file3.java

FILE1.java consists of main() in which f1(),f2() functions are present

FILE2.java consists of f1() function

FILE3.java consists of f2() function

All the files are fetch in to the complier, compiler converts .java files in .class files these object files consists of bytecode (file1.class, file2.class, file3.class consists of byte code).

JVM resides on RAM it consists of following components:

1,CLASS LOADER

2,BYTE CODE VERIFIER

3,EXECUTION ENGINE

CLASS LOADER: This component loads the class files in to the RAM.

BYTE CODE VERIFIER: It check the operands stack underflows, stack overflows due to these reason JAVA is secure.

EXECUTION ENGINE: It converts bytecode into machinecode using JUST IN TIME CODEGENERATOR (JIT CODE GENERATOR).

The machine code produced is executed by operation system and processor.
