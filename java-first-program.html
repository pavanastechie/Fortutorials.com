FIRST PROGRAM IN JAVA

To write a java program, we need an editor the most popular editors for creating a java program are:

1, NOTEPAD

2, NETBEANS IDE

3, ECLIPSE IDE

Among 3 editors notepad is the basic editor for the beginners of java programming.

STEP-1:

Open a notepad and write java code as follows:

class welcome {
public static void main(String[] args){
System.out.println("Welcome to FORTUTORIALS.COM");
}
}

STEP-2:

Save the notepad as welcome.java or "welcome.java" because class name and notepad name should be same.

Precautions while write a program:

1, S in String[] is always an uppercase(only S).

2, S in System.out is always uppercase (only S).

3, This is important precaution, notepad name and class name should be alike, uppercase is not same to lowercase, i.e. (welcome is not identical to WELCOME so, be careful).

STEP-3:

After completion of writing java code, we have to compile the code using CMD. Open cmd in windows, testing java .

STEP-4:

Identify the location of notepad file (welcome.java); suppose we saved our file in D-drive. Open D-drive in cmd write d: press enter, as below:

It opens fortutorials folder.

STEP-5:

Compilation of the java program, write javac (space) welcome.java press enter, as follows:

It shows error about System misspells as Syste. Then open welcome.java file modify the error and recompile again.

STEP-6:

After compilation next step is execution of the program. Write java (space)welcome in cmd and then press enter, as follows:

Welcome to FORTUTORIALS.COM is output of the program.

UNDERSTANDING JAVA PROGRAM

class welcome {
public static void main(String[] args){
System.out.println("Welcome to FORTUTORIALS.COM");
}
}

CLASS:

Class is instance of objects; class is place where we can declare the properties of elements on which we are writing program. Suppose, we are writing program for HONDA bikes. So, class can be consists of bikes CBR-1000, CBR-250, YZF-R6. Here all these models are instances means CBR-250 is one instance we can describe the properties (means tyre size, engine-size) of CBR-250 using object.

CLASS NAME

(welcome):

Class name is an identifier; it is used to name class for unique addressing of that class.

PUBLIC:

Public is reserved word in java, it is declare for classes, methods, variables by declaring public to the class, method and variable of that class is used for other class.

MAIN():

Java execution start for main() method in any program; afterwards main() calls other methods that are required to run your program. Generally main() resides in class itself.

STATIC:

Static keyword is used to access method without any class definition. Suppose, we declare static for main() method .Because java program start with main method(without using class definition) so we make main() as static.

VOID:

Void is the java reserved word declared for main() method. Because there is no return value (return null value) it doesn't return any value for calling function.

String[] args:

Here array is args. It takes array of string; when the method is called.

System.out.println(""):

This is the statement in java programming language in order to display the message on the screen. We have to declare the displaying statement in double quotes as "Welcome to FORTUTORIALS.COM" to display on screen.

Curly Braces {}:

These are used in the java program to denote the starting ({) of method, class and ending (}) of methods, class.

Semi colon (;):

Semicolon in java program denotes the termination of present line and switch to other line.
