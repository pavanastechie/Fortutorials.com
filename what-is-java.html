WHAT IS JAVA

Java is a programming language which is developed by James Gosling at sun microsystems, sun microsystems merged in to oracle in year 2010. First java software released in year 1995, its syntax is derived from c and c++. The important feature of java is that

write once, run anywhere(WORA)

It means that the java code is compiled once that can run in any system without recompilation. Java is most popular language particularly for client-server applications

Java is general purpose, concurrent, object-oriented, class-based programming language under GNU General Public License.
