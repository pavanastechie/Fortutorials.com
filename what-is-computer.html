What Is Computer

NOTE: binary code, machine language, 0's and 1's are same terms

For programmers (or) users computer is an electronic device which can understand only machine language (or) binary code (means 0's and 1's).

Example: Storing notepad document in memory, for user the notepad content is readable (or) understandable(English language) , but computer doesn't understand our English, French e.c.t. It can understand 0's and 1's so, notepad content is converted into machine language and then stored into computer memory whenever user open the notepad the binary code is converted to human readable format(converted to English language or other), this conversion is done by operating system (or) other software .

Suppose we open notepad and write "HELLO" and save the notepad document in memory, internally operating system (or) other software converts the HELLO into machine language and this machine code is stored in memory .

Machine code (or) binary code of HELLO is as follows:

H is 01001000

E is 01000101

L is 01001100

O is 01001111

HELLO- 0100100001000101010011000100110001001111 . To store HELLO word into memory, computer need this much of 0's and 1's.
