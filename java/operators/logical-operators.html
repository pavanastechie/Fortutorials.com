LOGICAL OPERATORS

Java consists of three logical operators; these logical operators are used to combine two or more relations. The combination of two or more relations is termed as a logical expressions or Compound relational expressions.

LOGICAL OPERATORS

OPERATORS DESCRIPTION SYNTAX & EXAMPLE
LOGICAL AND (&&) It checks the both relations, if two relations are true then condition is true. If one among them is false then total condition is false

a < b && b == c

if a lessthan b and bequal to c then the expression is true, if b notequal to c then expression is false.
LOGICAL OR (||) It checks the both relations, if one of the relation is true then condition is true. If all of them are false then total condition is false.

a < b || b == c

if a is less than b, b is equal to c any one relation among them is true, then total condition is true. If all are false then condition is also false.
LOGICAL NOT (!) It is not used frequently, it returns Boolean value such as true or false. It is used to reverse the value of the condition, if the condition is true, then it returns false and vice-versa.

b=!(15>10)

It returns b value as false. Here ! is used to denote NOT or Reverse the value which reverse the value of b from true to false.
