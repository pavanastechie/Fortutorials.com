Is greater than or equal to (>=)

Is greater than or equal to is an Relational operator used in the java programming language;it checks value of left operand is greater than or equal to right operand, if the condition is correct it return Boolean true else return false.

Is greater than or equal to Program

class greateq
{
public static void main(String[] args )
{
int a=15,b=10,c=10;
System.out.println("a is:"+a);
System.out.println("b is:"+b);
System.out.println("c is:"+c);
System.out.println("a is greater than or equal to b:"+(a>=b));//greater than
System.out.println("b is greater than or equal to c:"+(b>=c));//equal to
System.out.println("b is greater than or equal to a:"+(b>=a));
}
}

Output

D:\fortutorials>javac greateq.java
D:\fortutorials>java greateq
a is:15
b is:10
c is:10
a is greater than or equal to b:true
b is greater than or equal to c:true
b is greater than or equal to a:false
