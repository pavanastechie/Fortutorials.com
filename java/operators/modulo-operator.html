Modulo division (%)

Modulo operator is an arithmetic operator used in the java programming language; modulo operator is used to find remainder after the division of two numbers. Modulo operator is performed on any numeric data type supported in java.

Program

class modop
{
public static void main(String[] args )
{
int a=15,b=10,mod;
mod=a%b;//Modulo operator in between a, b
System.out.println("Modulo of a,b is:"+mod);
}
}

Output

D:\fortutorials>javac modop.java
D:\fortutorials>java modop
Subtraction of a,b is:5
