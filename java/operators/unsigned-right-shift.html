UNSIGNED RIGHT SHIFT(>>>)

Unsigned Right Shift is special case in right shift operator, when we shift bits in right shift operator it performs shift operation without affecting the integer sign. Unsigned right shift operator affects the sign of the integer. It replaces all 1's with 0's from left hand side during unsigned right shift operation.

Example

a=-1 (binary value of -1 is 11111111111111111111111111111111)

a >> 24

After right shift it produces a=-1, no change in bits.

a=-1(binary value of -1 is 11111111111111111111111111111111)

a >>> 24

a=255 (binary value of 255 is 00000000000000000000000011111111)

Program

class urshift{
public static void main(String[]args)
{
int a=-1;
System.out.println("a is " + a);
System.out.print("Binary value of a before right shift is ");
System.out.println(Integer.toBinaryString(a));
a=a >> 24 ;//right shift OPERATOR
System.out.println("a value after right shift is " + a);
System.out.print("Binary value of a after right shift is ");
System.out.println(Integer.toBinaryString(a));
System.out.print("Binary value of a before Unsigned right shift is ");
System.out.println(Integer.toBinaryString(a));
a=a >>> 24 ;//unsigned right shift OPERATOR
System.out.println("a value after Unsigned right shift is " + a);
System.out.print("Binary value of a after Unsigned right shift is ");
System.out.println(Integer.toBinaryString(a));
}
}
