PRE DECREMENT OPERATOR

The symbol of pre decrement operator is used in front of the variable as '--a'.

a=1-a;

--a means subtracting 1 from variable or operand.

Example

int a=2;

System.out.println("--a is "+ --a);

Compiler scan the --a from left to right first encounter -- so, it subtracts 1 from 'a' and then displays change a=1 value.

Program

class predec{
public static void main(String[]args)
{
int a=2;
System.out.println("--a is "+ --a);//++a first it decrements, then it display decremented value.
}
}

Output

D:\fortutorials>javac predec.java
D:\fortutorials>java predec
--a is 1
D:\fortutorials>
