ASSIGNMENT OPERATORS

Assignment operators are used for assigning value to variables. Assignment operator is denoted by the symbol =. Java consists of many short hand assignment operators.

Syntax of short hand assignment operator is as follows

Variable OP = EXPRESSION

OP is binary operator, Expression and variable.

Example

a=a+(b+c) short hand notation is a+=(b+c).

VARIABLE = a

OP = +

EXPRESSION = (b+c).

SHORT HAND NOTATIONS

SYMBOL NAME EXAMPLE
= SIMPLE ASSIGNMENT OPERATOR

a=25

Integer value 25 is assigned to variable a.
+= ADD AND ASSIGNMENT OPERATOR

a+=c

a=a+c is simply represented as a+=c.
-= SUBTRACT AND ASSIGNMENT OPERATOR

a- =c

a=a-c is simply represented as a- =c.
*= MULTIPLY AND ASSIGNMENT OPERATOR

a*=c

a=a*c is simply represented as a*=c.
/= DIVIDE AND ASSIGNMENT OPERATOR

a/=c

a=a/c is simply represented as a/=c.
%= MODULUS AND ASSIGNMENT OPERATOR

a%=c

a=a%c is simply represented as a%=c.
< < = LEFT SHIFT AND ASSIGNMENT OPERATOR

a< <=4

a=a< < 4 is simply represented as a < < =4.
> >= RIGHT SHIFT AND ASSIGNMENT OPERATOR

a> > =4

a=a> > 4 is simply represented as a> > =4.
&= BITWISE AND ASSIGNMENT OPERATOR

a&=c

a=a& c is simply represented as a&=c.
^ = BITWISE EXCLUSIVE OR AND ASSIGNMENT OPERATOR

a ^ =c

a=a ^ c is simply represented as a ^ =c.
|= BITWISE INCLUSIVE AND ASSIGNMENT OPERATOR

a|=c

a=a|c is simply represented as a|=c.

Program

class assign {
public static void main(String args[]) {
int a = 2;
int m = 0;
System.out.println("a is" + a );
System.out.println("m is" + m );
m = a;
System.out.println("simple assignment m = a = " + m );
m += a ;//ADD AND ASSIGNMENT OPERATOR
System.out.println("output of m += a or m= m+a is " + m );
System.out.println("m is " + m );
m -= a ;//SUBTRACT AND ASSIGNMENT OPERATOR
System.out.println("output of m -= a or m= m-a is " + m );
System.out.println("m is " + m );
m *= a ;//MULTIPLY AND ASSIGNMENT OPERATOR
System.out.println("output of m *= a or m= m*a is " + m );
System.out.println("m is " + m );
m/= a ;//DIVIDE AND ASSIGNMENT OPERATOR
System.out.println("output of m /= a or m= m/a is " + m );
System.out.println("m is " + m );
a = 3;
m = 4;
m %= a ;//MODULUS AND ASSIGNMENT OPERATOR
System.out.println("output of m %= a or m= m%a is " + m );
System.out.println("m is " + m );
m <<= 3 ;//LEFT SHIFT AND ASSIGNMENT OPERATOR
System.out.println("output of m <<= 3 or m= m<<3 is " + m );
System.out.println("m is " + m );
m >>= 3 ;//RIGHT SHIFT AND ASSIGNMENT OPERATOR
System.out.println("output of m >>= 3 or m= m>>3 is " + m );
System.out.println("m is " + m );
m &= a ;//BITWISE AND ASSIGNMENT OPERATOR
System.out.println("output of m &= a or m= m&a is " + m);
System.out.println("m is " + m );
m ^= a ;//BITWISE EXCLUSIVE OR AND ASSIGNMENT OPERATOR
System.out.println("output of m ^= a or m= m^a is " + m );
System.out.println("m is " + m );
m |= a ;//BITWISE INCLUSIVE AND ASSIGNMENT OPERATOR
System.out.println("output of m |= a or m= m|a is " + m );
}
}
