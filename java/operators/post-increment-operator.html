POST INCREMENT

In Post Increment symbol of increment operator is used after the variable as 'a++'

a++=a+1

a++ means adding 1 to variable or operand.

Example

int a=2;

System.out.println("a++ is "+ a++);

Compiler scan the a++ from left to right first encounter variable 'a' and then ++ change a=3 value in next step.

Program

class postinc{
public static void main(String[]args)
{
int a=1;
System.out.println("a++ is "+a++);// it show initial value of a, increment is effects from next line.
System.out.println("a is "+a);
}
}

Output

D:\fortutorials>javac postinc.java
D:\fortutorials>java postinc
a++ is 1
a is 2
D:\fortutorials>
