POST DECREMENT OPERATOR

In Post Decrement symbol of decrement operator is used after the variable as 'a--'.

a--=a-1;

a-- means subtracting 1 from variable or operand.

Example

int a=2;

System.out.println("a-- is "+ a--);

Compiler scan the a-- from left to right first encounter variable 'a' and then -- change a=1 value in next step.

Program

class postdec{
public static void main(String[]args)
{
int a=2;
System.out.println("a-- is "+a--);// it show initial value of a, increment is effects from next line.
System.out.println("a is "+a);
}
}

Output

D:\fortutorials>javac postdec.java
D:\fortutorials>java postdec
a-- is 2
a is 1
D:\fortutorials>
