LOGICAL OR (||)

Logical OR is logical operator used to join two or more relations, syntax of logical OR is ||. It checks the both relations, if one of the relation is true then condition is true. If all of them are false, then total condition is false.

Example

(a > b) || (b == c)

(a < b) || (b == c) && (c > a)

Program

class logor{
public static void main(String args[]) {
int a = 10;
int b = 5;
int c = 10;
System.out.println(" a is "+a);
System.out.println(" b is "+b);
System.out.println(" c is "+c);
System.out.println(" Statement(a > b) is "+(a > b));//first relation
System.out.println(" Statement(a == c) is "+(a == c));// second relation
System.out.println("Statement(a > b || a == c):a is greaterthan b or a is equal to c is " + (a> b || a==c) );// all relations are true
System.out.println("Statement(a> b || a != c):a is greaterthan b or a is not equal to c is " + (a> b&&a!=c) );// second is false
System.out.println("Statement(a < b || a == c):a is less than b or a is equal to c is " + (a < b || a==c) ); // first is false
System.out.println("Statement(a > b || a != c):a is less than b or a is not equal to c is " + (a < b || a != c) );// all are false
}
}
