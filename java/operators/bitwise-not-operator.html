BITWISE NOT (~)

Bitwise not is unary operator, it inverse all the value of operand. It is also called as Bitwise compliment; it performs operation between two integer values.

Integer values converted in to binary code (i.e. 0's and 1's). Then perform bit-to-bit NOT operation.

NOT: Suppose we perform NOT operation ON A, then output ~a is as follows

Truth Table OF NOT

A ~A
1 0
0 1
Example A=2 (binary code of 2 is 0010 taken last four bits, 32-bit binary value -00000000000000000000000000000010) then , ~A = -3 (binary code of -3 is 1101 taken last four bits,32-bit binary value -11111111111111111111111111111101)
A=2 ~A=-3
0 1
0 1
1 0
0 1

Program

class not{
public static void main(String[]args)
{
int a=2;
System.out.println("a is " + a);
a =~a ;//BITWISE NOT OPERATOR
System.out.println("output of ~a is " + a);
System.out.print("Binary value of ~a is ");
System.out.println(Integer.toBinaryString(a));
System.out.print("Binary value of a is ");
System.out.println(Integer.toBinaryString(~a));
}
}
