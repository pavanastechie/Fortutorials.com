LOGICAL NOT (!)

Logical NOT is logical operator used to reverse the value of relation, syntax of logical NOT is !. It is not used frequently, it returns Boolean value such as true or false. It is used to reverse the value of the condition, if the condition is true, then it returns false and vice-versa.

Example

b = !(15 > 10)

condition 15 > 10 is true but, ! Symbol makes as false.

Program

class lognot
{
public static void main(String[] args )
{
int a=15,c=10;
System.out.println("a is:"+a);
System.out.println("c is:"+c);
System.out.println("Statement(a > c): a is greaterthan c is "+(a > c));
System.out.println("Statement !(a > c): negation of a greater than c is "+!(a > c));
System.out.println("Statement(a < c): a is lessthan c is "+(a < c));
System.out.println("Statement !(a < c): negation of a less than c is "+!(a < c));
}
}

Output

D:\fortutorials>java lognot
a is:15
c is:10
Statement(a > c): a is greaterthan c is true
Statement !(a>c): negation of a greater than c is false
Statement(a < c): a is lessthan c is false
Statement !(a < c): negation of a less than c is true
