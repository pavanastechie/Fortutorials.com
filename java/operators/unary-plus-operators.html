Unary plus operator (+)

Unary plus operator is arithmetic operator used in java. Usually unary operator is used to denote the sign of the variable value. By default the variable value is positive.

int z=20;//is positive value

int x=+20//is also positive value

Unary plus operator Program

public class unaryplus{
public static void main(String args[])
{
int x=10;
int y=+10;
int z=-10;
int a=0;
System.out.print("sign of x is:");
System.out.println(Integer.signum(x));
System.out.print("sign of y is:");
System.out.println(Integer.signum(y));
System.out.print("sign of z is:");
System.out.println(Integer.signum(z));
System.out.print("sign of a is:");
System.out.print(Integer.signum(a));
}
}

Output

D:\fortutorials>javac unaryplus.java
D:\fortutorials>java unaryplus
sign of x is:1
sign of y is:1
sign of z is:-1
sign of a is:0
D:\fortutorials>

Signum is method in java.lang.Math, it is used to detect the sign of the variable and returns 1, 0, -1 as follows:

Different signs Return type values
If the variable sign is positive(+ve) 1
If the variable sign is zero 0
If the variable sign is negative(-ve) -1
