LEFT SHIFT OPERATOR(<<)

Left Shift Operator is one of the important bitwise operator; left shift operator is applied on the bits of the integer. The symbol of left shift operator is '<<'. It shifts the bits towards left hand side.

Variable << num

Variable are such as a,b so on

num specifies the number bits to be shift left.

Example

int a=1;

A<<2;

A is variable and 2 bits shift towards left.

Binary code of 1 is 0001.

Shifting '2' positions left, then the binary code of shifted value is 0100, its Decimal value is 4.

Program

class lshift{
public static void main(String[]args)
{
int a=1;
System.out.println("a is " + a);
System.out.print("Binary value of a before left shift is ");
System.out.println(Integer.toBinaryString(a));
a=a << 2 ;//left shift OPERATOR
System.out.println("a value after left shift is " + a);
System.out.print("Binary value of a after left shift is ");
System.out.println(Integer.toBinaryString(a));
}
}

Output

D:\fortutorials>javac lshift.java
D:\fortutorials>java lshift
a is 1
Binary value of a before left shift is 1
a value after left shift is 4
Binary value of a after left shift is 100
