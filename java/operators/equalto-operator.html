Is equal to Operator (==)

Is equal to operator is an Relational operator used in the java programming language;it is used to check whether the two variables are equal or not, if two variables are equal then return Boolean value true else return false.

Is equal to Program

class equal
{
public static void main(String[] args )
{
int a=10,b=10,c=15;
System.out.println("a is:"+a);
System.out.println("b is:"+b);
System.out.println("c is:"+c);
System.out.println("a is equal to b:"+(a==b));
System.out.println("a is equal to c:"+(a==c));
}
}

Output

D:\fortutorials>javac equal.java
D:\fortutorials>java equal
a is:10
b is:10
c is:15
a is equal to b:true
a is equal to c:false
D:\fortutorials>
