CONDITIONAL OPERATOR

Java includes special character pair ? : called ternary operator. This operator is used to construct conditional expressions, this conditional operator is used as substitution of if - else statement.

Syntax

expression1? expression2 : expression 3

Compiler first execute expression1 if it is true, then it evaluate expression2 as the value of the conditional expression.

Suppose expression1 is false, then it evaluate expression3 as the value of the conditional expression.

Example

b=8

c=9

x=(c>b) ? c : b

expression1: c>b

expression2: c

expression3: b

if expression1 c>b (9 > 8) is true

return expression2 i.e. c(9) as output

if expression 1 is false

return expression3 i.e. b(8) as output

Program

class cond{
public static void main(String[]args)
{
int a=2;
int b=3;
boolean c;
int result;
c= a < b;
result=c ?a:b;
System.out.println("the value of conditional operator c ? a : b is "+result);
}
}

Output

D:\fortutorials>javac cond.java
D:\fortutorials>java cond
the value of conditional operator c ? a : b is 2
