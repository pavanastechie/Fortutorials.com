Literals

Literals are the java tokens; they are used in the java program to represent fixed values.

A literal means a digit, text, character e.c.t.

EXAMPLE

int value=10;

In the above example the number 10 is a literal.

There are different types of literals

Floating-Point Literals

Character Literals

String Literals

Backslash Literals
