Decimal integer Literals

Decimal integer literals consists of digits 0 through 9, they consists of negative number and positive numbers.

Example

Positive numbers: 300, 597, 465, 0

Negative numbers: -589, -400, -898

Spaces between digits, commas, and non-digit characters are not permitted between digits.

Example

1 500

20,00

$1000($ is non-digit) are illegal numbers.

PROGRAM

public class declit {
public static void main(String[] args){
int val1=49;
int val2=-786;
System.out.println("DECIMAL INTEGER LITERAL POSITIVE VALUE = "+val1);
System.out.println("DECIMAL INTEGER LITERAL NEGATIVE VALUE = "+val2);
}
}

OUTPUT

D:\fortutorials>java declit
DECIMAL INTEGER LITERAL POSITIVE VALUE = 49
DECIMAL INTEGER LITERAL NEGATIVE VALUE = -786
