Octal integer Literals

An octal integer literal consists of any combination of digits 0 through 7, octal integer start with 0.

Example: 010, 1300L

PROGRAM

public class octlit {
public static void main(String[] args){
int val1=012;
int val2= 032;
System.out.println("OCTAL INTEGER LITERAL VALUE = "+ val1);
System.out.println("OCTAL INTEGER LITERAL VALUE = "+val2);
}
}

OUTPUT

D:\fortutorials>javac octlit.java
D:\fortutorials>java octlit
OCTAL INTEGER LITERAL VALUE = 10
OCTAL INTEGER LITERAL VALUE = 26
D:\fortutorials>
