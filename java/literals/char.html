Character Literals

A single character literal contains a single character enclosed with in a pair of single quote marks.

Example

'9', ' ', 'u'

Character literal'9' is not same as the number 9. ' ' is blank space and 'u' is a character.

PROGRAM

public class charlit {
public static void main(String[] args){
char val1='5';
char val2='c';
System.out.println("CHAR LITERALVALUE = "+ val1);
System.out.println("CHAR LITERAL VALUE = "+val2);
}
}

OUTPUT

D:\fortutorials>javac charlit.java
D:\fortutorials>java charlit
CHAR LITERALVALUE = 5
CHAR LITERAL VALUE = c
D:\fortutorials>
