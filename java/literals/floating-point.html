Floating-Point Literals

These quantities are represented by number and fractional part like 17.548 such numbers are called real (floating point) literals.

Example

1.367, 3.9847, -0.645

Where .367, .9847, .645 are fractional part.

We can write 215., .95, -.71 are valid real numbers.

Floating-Point Literals also expressed in exponential notation .for example, the value 9325.78 can be written as 9.32578e3 in exponential notation.e3 means multiply by 103.

We can represent 0.000067 can be written as 6.7E-5.

Note:

where we can use E (or) e.

PROGRAM

public class reallit {
public static void main(String[] args){
double val1=4.5678;
double val2=-6.5743;
System.out.println("FLOATING-POINT LITERAL POSITIVE VALUE = "+ val1);
System.out.println("FLOATING-POINT LITERAL NEGATIVE VALUE = "+val2);
}
}

OUTPUT

D:\fortutorials>java reallit
FLOATING-POINT LITERAL POSITIVE VALUE = 4.5678
FLOATING-POINT LITERAL NEGATIVE VALUE = -6.5743
