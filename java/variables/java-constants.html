JavaConstants

The constant is a variable whose value remains unchanged throughout the program; usually Java doesn't support constant type. But, by using some keywords (or) reserved words like public, static and final we can declare a constant in Java.

The constant name is an identifier; for naming constant we use uppercase and underscores ( _ ). We can initialize the constant anywhere in a program.

public static final int MAX_VALUE = 30;

above statement is the declaration of constant.

EXAMPLE:

MAX_VALUE=30;

Program

public class var {
public static final int MAX_VALUE = 30;
public static void main(String[] args){
for (int i = 0;i <= 3;i++){
int change=i;// i value change ten time due to for()
System.out.println(" MAX_VALUE value = "+ MAX_VALUE);
System.out.println("change value = "+change);
}
}
}

INTEGER CONSTANTS

An integer constant consists of sequence of digits. Integer constants are further divided in to:

Decimal integer

Octal integer

Hexadecimal integer

Decimal integer:

Decimal integer consists of digits 0 through 9, they consists of negative number and positive numbers.

Example:

Positive numbers: 300, 597, 465, 0

Negative numbers: -589, -400, -898

Spaces between digits, commas, and non-digit characters are not permitted between digits.

Example:

1 500

20,00

$1000($ is non-digit) are illegal numbers.

PROGRAM

public class jintcons {
public static final int DEC_VAL1 = 30;
public static final int DEC_VAL2 =-200;
public static void main(String[] args){
System.out.println("DECIMAL INTEGER CONSTANT POSITIVE VALUE = "+ DEC_VAL1);
System.out.println("DECIMAL INTEGER CONSTANT NEGATIVE VALUE = "+DEC_VAL2);
}
}

Hexadecimal integer:

A sequence preceded by 0x(or) 0X is considered as hexadecimal integer. They may also include alphabet A through F (or) a through f. A letter A through F represents the numbers 10 through 15.

Example:

0x2, 0X9f, 0x9F, 0X

Real Constants:

These quantities are represented by number and fractional part like 17.548 such numbers are called real(floating point) constants.

Example:

1.367, 3.9847, -0.645

Where .367, .9847, .645 are fractional part.

We can write 215., .95, -.71 are valid real numbers.

Real numbers also expressed in exponential notation .for example, the value 9325.78 can be written as 9.32578e3 in exponential notation.e3 means multiply by 103.

We can represent 0.000067 can be written as 6.7E-5.

Note: where we can use E (or) e.

PROGRAM

public class jintcons3 {
public static final double REAL_VAL1 = 7.856;
public static final double REAL_VAL2 =-3.67892345E10;
public static void main(String[] args){
System.out.println("REAL INTEGER CONSTANT POSITIVE VALUE = "+ REAL_VAL1);
System.out.println("REAL INTEGER CONSTANT NEGATIVE VALUE = "+REAL_VAL2);
}
}

Character constants:

A single character constant contains a single character enclosed with in a pair of single quote marks.

Example:

'9', ' ', 'u'

Character constant '9' is not same as the number 9. ' ' is blank space and 'u' is a character.

PROGRAM

public class jchars {

System.out.println("CHAR CONSTANT VALUE = "+ CHAR_VAL1);
System.out.println("CHAR CONSTANT VALUE = "+CHAR_VAL2);
}
}

CHAR CONSTANT VALUE = 5

String constant:

Backslash character constants:

Java supports some special backslash constants that are used in output methods they are given below:

\t- it is horizontal tab to maintain space between characters.

\b-insert a backspace in the text.

\n-insert a newline

\r-insert carriage return

\f-insert a form feed

\'- insert a single quote character

\"- insert a double quote character

\\ -insert a backslash character.

PROGRAM

public class backslash {
public static void main(String[] args){
int number=10;
System.out.println("tab\tvlaue:"+number);
System.out.println("back space in text as\b:"+number);//s is removed in as due to back space
System.out.println("new line in program \n as:"+number); //as is printed in new line
System.out.println("form feed in text as\f:"+number);// a space symbol in statement
System.out.println("singlequote in text\'s as"+number);// \' inserts one single quote
System.out.println("double quote in text\"hello\"as"+number);//\" inserts one double quote
System.out.println("back space in text \\ as:"+number);//inserts backslash in text
System.out.println("back space in text \r as:"+number);// prints as:+number at starting of string
}
}
