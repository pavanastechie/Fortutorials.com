Instance variables

Instance variable are non-static type because, they are declares without static keyword because, the value is changed for every object. Instance variables are declared inside the class, but outside the methods, constructors and blocks. They are created when the object of the class is created and destroyed when object is destroyed.The instance variables are visible for all methods, constructors and block in the class.Instance variable can have default value.

Program

class yamaha{
int speed;
}
class testspeed
{
public static void main(String args[])
{
int area;
yamaha test1= new yamaha();
test1.speed=100;
System.out.println("when we check speed before 20seconds of yamaha bike "+test1.speed+" km/hour");
yamaha test2=new yamaha();
test2.speed=80;
System.out.println("when we check present speed of yamaha bike "+test2.speed+" km/hour");
}
}

OUTPUT

when we check present speed of yamaha bike 80 km/hour
