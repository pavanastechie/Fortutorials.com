Class Variables (static):

The class variables are declared within the class and outside of the methods, constructor, or blocks. They are declared with keyword static, this tells the compiler that there is exactly one copy of this variable in existence. Additionally the final keyword is added to make the variable constant value.

Static int number=10;

Static final int number=10; // final is added to the variable to make it constant value.

Final is declared in the variable declaration to make the variable constant throughout the class.

PROGRAM

public class yamaha{
static final int gears= 6;// gear must be constant for same model bikes
public static void main(String[] args){
yamaha R1 = new yamaha();//R1 is yamaha model
System.out.println("No. of gears for first yamaha R1 : " + R1.gears);
yamaha anotherR1 = new yamaha();//Another R1 bike
System.out.println("No. of gears for second yamaha R1 : " + anotherR1.gears);
System.out.println("No. of gears for both the R1 should be same because they are same models so, we use class variables for fixed values.");
}
}

OUTPUT
