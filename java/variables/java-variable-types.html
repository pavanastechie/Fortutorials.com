Types of Variables

Variable is change throughout execution of the program, it takes different values at different times during execution of the program.

Example

If we take average as variable then, it changes with the given set of numbers.

a=3, b=5 then average is 3+5/2=4.

if a=4, b=8 then, average is 4+8/2=6.

So, that average changes for different set of a, b.

Program

public class var1 {
public static void main(String[] args){
int change=10;
System.out.println(" MAX_VALUE value = "+ change);
change=50;// no need of int change=50
System.out.println("change value = "+change);
}
}

OUTPUT

D:\fortutorials>javac var.java
D:\fortutorials>java var
MAX_VALUE value = 10
change value = 50
D:\fortutorials>

Change is variable it change two time in the program for initialization we use int change=10, second time no need of int change=50(change is already initialized) just specify change=50.
