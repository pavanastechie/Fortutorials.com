JAVA VARIABLES

Variable is an identifier, they are the names of the storage locations.

Variable declaration does following things:

1, It tells the compiler the variable name.

2,It specifies what type of data the variable will hold.

3,The place of declaration decides the scope of the variable.

Variable must be declared before it is used in the program. Variable can store a value of any data type.

The declaration is as follows:

int x;

double y,z,u;

Multiple variables are separated by commas and ended by semi colon.

We can give values to variable in 3 ways

Assignment Statement

Command line

Read Statement
