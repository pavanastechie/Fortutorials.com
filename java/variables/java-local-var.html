Local variables

Local variables are declared in the methods constructors, or blocks. Local variables are visible to the methods in which they are declared. Variable will be destroyed once it exits the method, constructor or block. These values are required to be initialized before using them. These variables don't take default values if not initialized.

int num=7;// declaration of local variables.

PROGRAM

public class local {
private void print (){
int i = 1; // local variable
System.out.println("local variable is "+i);
}
public static void main(String[] args) {
local test = new local();
test.print();//method accessing.
}
}

OUTPUT

D:\fortutorials>javac local.java
D:\fortutorials>java local
local variable is 10
