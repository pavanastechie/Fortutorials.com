Read Statement

Variable values can be read from the keyboard while compiling the program.

PROGRAM

import java.io.DataInputStream;
class key{
public static void main (String args[])
{
DataInputStream in=new DataInputStream(System.in);
int intnumber=0;
try{
System.out.println("ENTER THE INTEGER :");
intnumber=Integer.parseInt(in.readLine());
}
catch (Exception e)
{
}
System.out.println("int number is:"+intnumber);
}
}

OUTPUT

D:\fortutorials>javac key.java
Note: Reading.java uses or overrides a deprecated API.
Note: Recompile with -Xlint:deprecation for details.
D:\fortutorials>java key
ENTER THE INTEGER :
2
Int number is:2
D:\fortutorials>

In the above program we declare to variable

intnumber(integer type)

By using DataInputStream class we are invoking in.readLine() to read the given int and We use Integer.parseInt(in.readLine()); after giving int value from keyboard it is in the form of string using this line string is converted in to int.
