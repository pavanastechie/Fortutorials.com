Parameters

Variables are declared in the parameters of methods. They consist of return type in order to return the values.

Wantdata(int x, int y);//calling function

Wantdata(12,30);//called function

PROGRAM

class parameter{
int first,second;
void wantdata (int i, int j)// parameters variables
{
first=i;
second=j;
}
}
class numbers
{
public static void main(String args[])
{
int adds;
parameter p=new parameter();
p.wantdata(10,30);// parameters i,j are initializing
System.out.println("ADDITION OF FIRST,SECOND IS :"+p.first);
System.out.println("ADDITION OF FIRST,SECOND IS :"+p.second);
}
}
