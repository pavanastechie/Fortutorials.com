Command Line

Command line argument is the popular method for giving input to the variable.

We can give the variable value while running the program as:

Java file_name value1 value2 //spacing between values.

Javac arg.java//after compilation

Java arg 1 2 3

Whatever we are given as value it is taken as string

Example we have given

Java arg 1 2 3

1, 2, 3 are taken as string values not as int value because seen the main method (String args[]) args value is string, so compiler takes as strings. Whatever we are giving it is taken in to args[i], i value is incrementing.

We can declare args range using:

args.length=3;// this is string length.

So, we used the statement

Integer.parseInt(args[i])// for conversion of string value to int values.

Program

public class arg{
public static void main(String args[]){
int num1 = Integer.parseInt(args[0]);
int num2 = Integer.parseInt(args[1]);
int num3 = Integer.parseInt(args[2]);
System.out.println("The number is: " + num1);
System.out.println("The number is: " + num2);
System.out.println("The number is: " + num3);
System.out.println("sum of num1 and num2");
int c=num1+num2;
System.out.println("sum is:"+c);
}
}

Output

D:\fortutorials>javac arg.java
D:\fortutorials>java arg 1 2 3
The number is: 1
The number is: 2
The number is: 3
sum of num1 and num2
sum is:3
D:\fortutorials>

If we directly add numbers with out Integer.parseInt(args[i]);

Program

public class arg1{
public static void main(String args[]){
System.out.println("The number is: " + args[0]);
System.out.println("The number is: " + args[1]);
System.out.println("sum of args[0] and args[1]");

System.out.println("sum is:"+c);
}
}

Output

found : java.lang.String
required: int int c=args[0]+args[1];
^
1 error
