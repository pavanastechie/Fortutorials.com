Assignment statement:

Int x=10; //= is assignment statement

It is called as initialization, we can initialization at starting and where ever you want in the program.

Program

public class varint{
public static void main(String args[]){
int x;
int y=10;//initializing y at starting
System.out.println("y value is :"+y);
x=20;// initialization whenever necessary.
System.out.println("x value is:"+x);
}
}

OUTPUT

D:\fortutorials>javac varint.java
D:\fortutorials>java varint
y value is :10
x value is:20
