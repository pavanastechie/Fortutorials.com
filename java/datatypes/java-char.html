Char

Char is a different from other data types, The size of char is 2 bytes. Byte accepts the value in ranges between 0 to 65,535. Char can store any character but, the term is in between single quotes (' ').

Example:

char grow = 'A'

TYPE SIZE MINIMUM VALUE MAXIMUM VALUE
char two bytes 0 65,535

Char can store any value it is a digit, letter, space, tab e.c.t. but reads only single character.

PROGRAM

public class jchar {
public static void main(String[] args) {
char num = 'e';// char value must be in single quotes
char number = 65;// if we enter number char gives ASCII code of the number
char ascii = 66;
char i='6';
char j= ' " ';
System.out.println("Value of char variable num is :" +num);
System.out.println("ASCII value of char variable number is :"+number);
System.out.println("ASCII value of char variable ascii is :"+ascii);
System.out.println("Value of char variable i is :" +i);
System.out.println("Value of char variable j is :" +j);
}
}
