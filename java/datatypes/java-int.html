Int

It is an integer type, it stores only signed values they can be positive (or) negative. Int accepts the value in ranges between -2,147,483,648 to 2,147,483,647. Size is 4 byte. We declare int because the time required for execution is less compared to other data types.

Example:

int 3000 takes less time for manipulation compared to long 3000 because, the memory size of int is less compared to long.

Memory size of int ranges from -2,147,483,648 to 2,147,483,647.

Memory size of long ranges from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807.

When we declare int, compiler arranges -2,147,483,648 to 2,147,483,647 spaces in memory for our identifier.

TYPE SIZE MINIMUM VALUE MAXIMUM VALUE
int four bytes -2,147,483,648 2,147,483,647

Program

public class Type{
public static void main(String[] args)
{
int max=2147483647 ;
int min=-2147483648 ;
int num=7988000;
System.out.println("MAXIMUM VALUE OF INT DATA TYPE IS:" +max);
System.out.println("MINIMUM VALUE OF INT DATA TYPE IS:" +min);
System.out.println("VALUE OF INT DATA TYPE IS:" +num);
}
}

If we exceed the range of int data type it gives compiler error, see below program:

public class Types{
public static void main(String[] args)
{ int num=2147483648;
System.out.println("VALUE OF INT DATA TYPE IS:" +num);
}
}

OUTPUT:
