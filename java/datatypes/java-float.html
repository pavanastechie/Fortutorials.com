Float

It is a floating-point type, its size is 4bytes. float accepts the ranges between 1.40129846432481707e-45 to 3.40282346638528860e+38.

It deals with fractional point in integer type we miss value after point using float we can get fractional part.

Float is single-precision numbers while the double is double-precision number. Compiler treat Floating point numbers as double-precision. To force them to be in single-precision mode ,we must append F(or)f.

Example:

2.45968 is number we need to show by using int 2.45968 we get only 2 it remove .45968.

By declaring float 2.45968f we can show complete number.

PROGRAM

class fly {
public static void main(String[] args){
int a=8;
int b=3;
int c=a/b;
float g=8f;// float value is always extended with F(or)f
float h=3f; // float value is always extended with F(or)f
float j=g/h;
System.out.println("int number:"+c);
System.out.println("float number:"+j);
}
}

Using float we can get whole number + fractional part as 2.6666667.

If we exceed the range of float data type it gives compiler error, see below program:

public class flys{
public static void main(String[] args)
{
float max=34028234663852886000000000000000000000000f;//added one zero extra

System.out.println("MINIMUM VALUE OF FLOAT DATA TYPE IS:" +min);
