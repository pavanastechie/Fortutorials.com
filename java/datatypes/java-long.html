Long

It is an integer type, it stores only signed values they can be positive (or) negative. long accepts the value in ranges between -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807.

Size is 8 byte. To declare integers long by appending the letter L (or) l

Example:

1345L (or) 1345l

Memory size of long ranges from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807.

When we declare long, compiler arranges -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807. spaces in memory for our identifier.

TYPE SIZE MINIMUM VALUE MAXIMUM VALUE
long eight bytes -9,223,372,036,854,775,808 9,223,372,036,854,775,807

Program

public class Rock{
public static void main(String[] args)
{
long max=9223372036854775807L;
long min=-9223372036854775808l;
long num=9223372036854775800L;
long num1=2147483647; //integer max value so no L(or)l
long num2=2147483649L;// exceeds integer range so we have to use L(or)l
System.out.println("MAXIMUM VALUE OF LONG DATA TYPE IS:" +max);
System.out.println("MINIMUM VALUE OF LONG DATA TYPE IS:" +min);
System.out.println("VALUE OF LONG DATA TYPE IS:" +num);
System.out.println("VALUE OF LONG DATA TYPE IS:" +num1);
System.out.println("VALUE OF LONG DATA TYPE IS:" +num2);
}
}

In the above program we can see the maximum and minimum ranges of the long data type so that we can declare an value in between-9223372036854775808 to 9223372036854775807 but, extends with L(or)l to indicate it is long data type example: 9223372036854775567L we should not exceed -9223372036854775808 (or) 9223372036854775807.

We can declare the integer range without using L(or)l , if we exceed integer range we have to use L(or)l.

If we exceed the range of long data type it gives compiler error, see below program:

public class Rocks{
public static void main(String[] args)
{
long num=9223372036854775808L;
System.out.println("VALUE OF long DATA TYPE IS:" +num); }
}
