DATA TYPES

Data types specify the size and type of values that can be stored, number (or) digit and string given in java needs some memory while executing, this size is specified by using data types, data types in java divided into two types:

Primitive (built-in type)

Reference (non-primitive type)

Primitive data types:

All the primitive data types are predefined (keywords) they are not defined by user, there are 8 types:

byte

short

int

long

float

double

char

boolean

Reference data type:

These are not keywords they are nothing but identifiers created by user. Reference types are references (or) used to access an object.

There are 3 types:

class

array

interface
