Double

It is a floating-point type, its size is 8bytes. Double accepts the ranges between 4.94065645841246544e-324 to 1.79769313486231570e+308.

Double is used when we need greater precision then float, actual it is a double-precision. All mathematical functions, such as sin, cos and sqrt returns double type values.

TYPE SIZE MINIMUM VALUE MAXIMUM VALUE
double eight bytes 4.94065645841246544e-324 1.79769313486231570e+308

We can declare the double with extension d(or)D.

PROGRAM

public class Rdouble{
public static void main(String[] args)
{
double min= 4.94065645841246544e-324;
double max=1.79769313486231570e+308d;
double m=1.79e-50d; // Range higher then float
System.out.println("MAXIMUM VALUE OF DOUBLE DATA TYPE IS:" +max);
System.out.println("MINIMUM VALUE OF DOUBLE DATA TYPE IS:" +min);
System.out.println("VALUE OF DOUBLE DATA TYPE IS:" +m);
}
}

If we exceed the range of double data type it gives compiler error, see below program:

public class Rdouble1 {
public static void main(String[] args)
{
float max=1.79769313486231570e+309d ;//added one zero extra(308 to 309)
float min=1.89769313486231570e+308d ;//changed 1.7 to 1.8
System.out.println("MAXIMUM VALUE OF FLOAT DATA TYPE IS:" +max);
System.out.println("MINIMUM VALUE OF FLOAT DATA TYPE IS:" +min);
}
}
