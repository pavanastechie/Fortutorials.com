Boolean

Boolean is different from all the data types; it has only two values true and false used when there is two choices i.e. true (or) false. This data type represents one bit of information.

PROGRAM

public class jbool{
public static void main(String[] args)
{
boolean i=true ;
boolean j=false;
if(i==true)
{
System.out.println("i is - " +i);
}
if(j==false)
{
System.out.println("j is - " +j);
}
if(j==true)
{
System.out.println("j is - "+j);
}
else
{
System.out.println("j is - "+j);
}
}
}

In the above program we given two values i=true, j=false

We used the Boolean type in if condition if(i==true) here true is Boolean type.

If Boolean type i is equal to true(another Boolean type) the prints i is true.

In if(j==true) condition j is not equal to true so, it skip if condition(System.out.println("j is - "+j). goes to else part and prints j is - false.
