Features Of Java

JAVA AS A GENERAL PURPOSE LANGUAGE, INTERNET

A programming language whose use is not restricted to a particular type of computer or a specialized application is GENERAL PURPOSE LANGUAGE. Java is extending the limits of the Web, so it is called as general-purpose language, It is a huge programming language which is not confined to machine applications only. Java is more than just a tool for building transportable multimedia applications. Due to its diversity, it is a platform independent language.

Java is internet language because it supports

Write once-run anywhere

It supports internet protocols such as HTTP, FTP, IP, TCP-IP, DNS, SMTP E.C.T…………

Support of XML Parsing, Web Services, Mobile Devices, HTML E.C.T…………

CASE SENSITIVE:

Case sensitive is the property used by the computer to compare to word. Java is case sensitive language in which capital letter word differ from small letter word.

Example:

helloworld is different when compared to HelloWorld or HELLOWORLD

STRUCTURE OF JAVA

Before going to programming with java we have to understand the structure of java

Consider, java 7.0 it consists of two folders

JDK 1.7

JRE 7(1.7)

JDK 1.7:

JDK is abbreviated as Java Development Kit, JDK is used to develop the java program and to run the java programs it consists of following directories

BIN DIRECTORY

DB DIRECTORY

INCLUDE DIRECTORY

JRE DIRECTORY

LIB DIRECTORY

BIN

Bin directory consists of tools, these tools are used to compile, execute, debug, Java Debugger e.c.t

DB

DB is a relational database management system that is based on the Java and SQL. It deals with database.

INCLUDE

 It contains all header files like for 'C' programming language this directory content helps you to enable C language code into a Java programs.

JRE

Java Runtime Environment (JRE) is use to run the java, comparing JRE with JDK, here JDK is used develop and run the java files, where JRE is used to run java file.

LIB

Code libraries, property settings, and resource files used by the Java runtime environment. This is most important directory.

README.HTML

This file is browser supported file it will gives the complete details of JDK.

SRC.ZIP

This is the ZIP file usually it consists of source code of JDK.
